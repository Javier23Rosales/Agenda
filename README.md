⚡ Ultra Productivity - Focus Alarm

Una agenda diaria de alto rendimiento diseñada en React y Tailwind CSS con enfoque en la productividad diaria y alarmas visuales/sonoras.

🚀 Características

Agenda Diaria: Planificación hora por hora.

Sistema de Alarmas: Notificaciones push y sonidos integrados.

Diseño Premium: Interfaz Dark Mode con estética Glassmorphism.

Offline First: Funciona sin internet una vez descargada y guarda datos en el navegador (localStorage).

🛠️ Instalación y Uso

Descarga el archivo agenda.html.

Ábrelo en cualquier navegador moderno.

Para móviles: Selecciona "Añadir a la pantalla de inicio" para usarla como una aplicación nativa.
